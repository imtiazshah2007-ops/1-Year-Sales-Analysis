This is a complete, production ready version of the classic sales data EDA project.

This project takes raw combined sales order data, cleans it, performs feature extraction and generates a full set of standard business intelligence visualisations. This version fixes the extremely common annoying issue of unreadable scientific notation that plagues almost every other version of this analysis online.

What the project does:

Automatically cleans and converts raw input CSV data
Extracts month, hour, day and city dimensions
Generates and exports 6 high resolution 300DPI graphs:
Monthly sales trend
Best selling products by quantity
Revenue generated by product
Sales breakdown by city
Order volume by hour of day
Daily sales trend
Calculates additional metrics including average order value and top 5 highest value orders
All axis labels and values are formatted to display as $M or $K, there is no scientific notation anywhere.
